# Andy Portfolio — React + Vite + Tailwind CSS

This is a personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**.  
It showcases projects, skills, and experience in a clean, responsive UI.

---

## Tech Stack

- **React** — UI library
- **Vite** — Fast development & build tool with HMR
- **Tailwind CSS** — Utility-first CSS framework
- **ESLint** — Code quality & linting

---

## Development Setup

This project is based on the Vite React template, providing:
- Hot Module Replacement (HMR)
- Fast refresh using official Vite plugins

### Official Plugins
- `@vitejs/plugin-react` — Uses Babel for Fast Refresh
- `@vitejs/plugin-react-swc` — Uses SWC for faster builds
